# Mini Game Project (Tic Tac Toe)

## Objective

To create a console-based Tic Tac Toe game in C++ that demonstrates the use of loops, arrays, functions, and conditional statements.

## Features

* Two-player gameplay
* Dynamic game board display
* Win and draw detection
* Input validation
* Replay option

## Technologies Used

* C++
* Arrays
* Loops
* Functions
* Conditional Statements

## Expected Outcome

A functional Tic Tac Toe game that allows players to compete interactively while showcasing core programming concepts and game logic implementation.
